# Data Science Notebooks

## Get Started
#### Ubuntu
```
sudo apt-get install python-pip python-dev
sudo pip install numpy pandas scikit-learn seaborn jupyter
git clone https://github.com/bharatbgarg4/data-science-notebooks.git
cd data-science-notebooks
jupyter notebook
```

#### Windows
```
Install python
https://www.python.org/ftp/python/3.7.1/python-3.7.1-amd64.exe

Install Anaconda
https://repo.anaconda.com/archive/Anaconda3-5.3.0-Windows-x86_64.exe

Install missing libraries (Optional)
conda install seaborn

Download and extract notebooks
https://github.com/bharatbgarg4/data-science-notebooks/archive/master.zip

Open Anaconda navigator, launch Jupyter
```
